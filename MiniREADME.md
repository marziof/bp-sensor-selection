## Small Readme for Sensor Selection

## Content
- `sim_pipeline.py`: Main script to run the simulation pipeline for sensor selection. It generates synthetic data, applies different sensor selection methods, evaluates their performance, and saves the results.
- `results.ipynb`: Jupyter notebook for analyzing and visualizing the results of the sensor selection methods. It includes functions to plot comparisons of different methods based on various metrics.
- results folder: This folder will contain the CSV files generated by `sim_pipeline.py` with the results of the simulations for different parameter settings. Each file is named according to the parameters used in the simulation (e.g., number of sensors, delta, T_max, etc.).
- `greedy_algo.py`: Script for performing exact greedy subset selection + one by one + with fixed subset size - used in simulation pipeline
- `sim_helpers.py`: Helper functions for the simulation pipeline, including graph generation, dynamic process simulation (SI/dSIR).
- `metrics.py`: Functions to compute evaluation metrics such as Overlap, MO...
- `sensor_selection.py`: static sensors selection for BP
- `plot_helpers.py`: Helper functions for plotting results and comparisons of different methods.
